# Calculadora

Calculadora simples em Swift para iOS

#### Principais Recursos

- Função de Adição
- Função de Subtração
- Função de Multiplicação
- Função de Divisão
- Função de Porcentagem
- Função C
- Suporte a contas com vírgula



#

# Xcode 9.2:

- Esse software foi desenvolvido utilizando a IDE Xcode 9.2 



#

#### Melhorias Previstas
